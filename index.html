Google Tag Gateway & Consent Mode v2 Demo

Welcome! This static website is a simplified environment configured for testing and demonstrating Google measurement setup.

Google Tag Gateway (GTG)

Google Tag Gateway is set up on this website using a Cloudflare Snippet. It routes all tagging requests sent to /metrics/* through your first-party domain, proxying them to Google serving hosts and forwarding visitor geolocation headers.

Tag Container

The container GTM-NNCLVZ3 is configured using server-side tagging. It receives event data from the website first-party endpoint, processes it, and routes it to destinations.

Consent Mode

Google Consent Mode v2 is configured using the open-source Klaro consent manager. On initial page load, default consent is set to denied. User consent settings are dynamically mapped to Google Consent parameters. When accepted, the consent state updates instantly in GTM.
