Understanding Google Tag Gateway (GTG)

Historically, the Google Tag loads directly from www.googletagmanager.com/gtm.js or www.google-analytics.com/analytics.js. While this works fine, modern privacy architectures and strict security headers benefit significantly from routing these scripts through a first-party subdomain or path.

Key Advantages of first-party loading:

  • Security: Restricts the number of third-party domains in your site's Content Security Policy (CSP).
  • Reliability: Reduces block rates from standard client-side script filtering that specifically target third-party script hostnames.
  • Attribution: Keeps measurement signals resilient against third-party cookie constraints by associating cookies directly with the root domain.

The Cloudflare Snippet / Worker Proxy

Under the hood, this website is configured with a Cloudflare Snippet (or Worker) that matches the path /metrics/*. When a user loads this path, Cloudflare intercepts the request and forwards it to Google's First Party Serving endpoints: 

export default {
  async fetch(request) {
    const url = new URL(request.url);
    
    // Rewrite target to point to Google Tag Gateway endpoint
    url.hostname = "GTM-NNCLVZ3.fps.goog";
    
    const newRequest = new Request(url, request);
    newRequest.headers.set('X-Gtg-Implementation', 'Snippet');
    
    return await fetch(url, newRequest);
  }
};

Crucially, Cloudflare passes the geolocation headers (CF-IPCountry, CF-Region, etc.) which Google uses to determine user regions and apply region-specific consent rules (such as EEA regulations).

Advanced Consent Mode v2 Handshake

Advanced Consent Mode operates as follows:

  1. Initial page load hits. Consent defaults are immediately set to denied.
  2. GTM loads from /metrics/gtm.js. Because consent is denied, no tracking cookies are read or set, but cookieless pings are sent.
  3. Klaro loads, displays the notice, or reads existing cookie state.
  4. If user accepts, gtag('consent', 'update', { ...: 'granted' }) is triggered. Tags instantly change their behavior, loading cookies and resuming normal tracking.
